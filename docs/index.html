Tinybucket

A Ruby client library for Bitbucket REST API v2 with OAuth Authentication.

Download this project as a .zip file Download this project as a tar.gz file

Introduction

Tinybucket is a ruby wrapper library to use Bitbucket REST API v2 with OAuth Authentication.

This gem is inspired by vongrippen/bitbucket

WARNING

This gem is under development. SourceCodes are may be changed at any time.

Features

This gem has these features. See the README.md about detail.

Authentication

See sample code.
  • OAuth Authentication only

Teams

See sample code.
  • Get the team.
  • Iterate members of the team.
  • Iterate followers of the team.
  • Iterate following of the team.
  • Iterate repositories of the team.

Users

See sample code.
  • Get the user.
  • Iterate followers of the user.
  • Iterate following of the user.
  • Iterate repositories of the user.

Repositories

See sample code.
  • Iterate public Repositories
  • Iterate the repositories owned by the specified owner

Repository

See sample code.
  • Get the repository.
  • Get watchers of the repository.
  • Get forks of the repository.

PullRequest

See sample code.
  • Iterate PullRequests of the repository.
  • Get the PullRequest of the repository.
  • Iterate Commits of the PullRequest.
  • Approve the PullRequest.
  • Unapprove the PullRequest.
  • Get diff text of the PullRequest.
  • Iterate Comments on the PullRequest.
  • Get the Comment on the PullRequest.

Commits

See sample code.
  • Iterate Commits of the repository.
  • Get the Commit of the repository.
  • Iterate Commit Comments of the commit.
  • Get the Commit comment of the commit.

Branch Restrictions

See sample code.
  • Iterate Branch Restrictions of the repository.
  • Get the branch restriction of the repository.

diff

See sample code.
  • Get a diff text of the commit or branch.
  • Get a patch text of the commit or branch.
